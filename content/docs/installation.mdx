## 安装包

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
    <Tab value="npm">
    ```bash
    npm install @json-render/core @json-render/react
    ```
    </Tab>
    <Tab value="pnpm">
    ```bash
    pnpm add @json-render/core @json-render/react
    ```
    </Tab>
    <Tab value="yarn">
    ```bash
    yarn add @json-render/core @json-render/react
    ```
    </Tab>
    <Tab value="bun">
    ```bash
    bun add @json-render/core @json-render/react
    ```
    </Tab>
</Tabs>

## 同伴依赖

json-render 需要以下同伴依赖：

- `react` ^19.0.0
- `zod` ^4.0.0

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
    <Tab value="npm">
    ```bash
    npm install react zod
    ```
    </Tab>
    <Tab value="pnpm">
    ```bash
    pnpm add react zod
    ```
    </Tab>
    <Tab value="yarn">
    ```bash
    yarn add react zod
    ```
    </Tab>
    <Tab value="bun">
    ```bash
    bun add react zod
    ```
    </Tab>
</Tabs>
